I am the page that loads first. Please load a different page here. If you want to load a certain page when this page loads, do this: 
function init()
{
	setBuffer('loadTo','page2.html')
}
onload = init
That way you can load any page you want.

Each page that is loaded must have this in the header: 

<script src="seBuffer.js" type="text/javascript"></script>
<script type="text/javascript">
onload = init
</script>
