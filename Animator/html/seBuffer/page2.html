One thing to note: If you are using scripts, you have to load them on the actual page, not in the separate HTML files. Since the document is just grabbing the content of the body, it will ignore all the <script> tags and whatnot.

If you want do load the scripts in the actual page you load, then you could refence the script like so:

parent.frames['seBufferBox'].meFunction(). Hope that makes sense.
